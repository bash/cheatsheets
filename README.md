### Email max length

According to [https://www.ietf.org/rfc/rfc2821.txt "4.5.3.1 Size limits and minimums"](https://www.ietf.org/rfc/rfc2821.txt) the maximum length of an email address is 320 characters (local part: 64, domain: 255, @: 1)
